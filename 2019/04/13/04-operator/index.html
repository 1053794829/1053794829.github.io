Python支持的运算符：数学运算符，比较运算符，逻辑运算符，赋值运算符

1.数学运算符：+(加)，-（减），*（乘），/（除）,//（整除）,%（取余）,**（幂运算）

前四个加减乘除和数学中的一样

print(2+4) # 6
print(4-1) # 3
print(2*5) # 10
print(7/2) # 3.5
print(6/2) # 3

//(整除） 求商，只保留整数部分

print(7//2) # 3
print(4.2//2) # 2

%（取余） 求余数

print(7%3) # 1

xy,求x的y次方

print(2**3) # 8
