#day 2 总结

#1、常用的快捷键

Ctrl+s 保存

Ctrl+/ 在开头添加注释和取消注释

Ctrl+b 执行

Ctrl+shift+z 反撤销

Ctrl+x 剪切

Ctrl+f 查找

按住shift点鼠标 选中范围

#2、基础语法

1.注释

代码中不参与执行的说明性文字

多行注释三个双引号（”””）之间或者三个单引号（’’’）之间

2.标识符

程序中专门用来命名的

Python中标识符的要求：所有标识符是由字母、数字和下划线组成，并且数字不能开头

Python3.x中标识符包含了中文

变量名=值

##3.缩进

###一行代码前的空格叫缩进，Python中每行的缩进个数是有规范的

##4.关键字

###系统保留下来的具有特殊功能或特殊意义的标识符

5.常用的数据类型

int 整型（整数）

float 浮点型（小数）

str 字符串（文本，放在单引号’’或者双引号之间””)例如：’afsa’、’地方’、’123’

bool 布尔（只有True（真，成立）和False（否，不成立）两个值）

list 列表

dict 字典

set 集合

tuple 元祖、迭代器、生成器

##6.两个常用的内置函数：print（）、input（）

输出函数：print（），在控制台输出括号里面的内容

输入函数：input（），从输入设备获取输入的内容到程序中

输入结果=input（提示信息）

当程序执行到input函数时，程序会停下来等待输入完成后才会接着往下执行

练习：写程序用来收集一个人的姓名和年龄，并且将收集到的结果打印出来

1.什么是变量

变量是用来在内存中开辟空间保存数据的

2.怎么声明/定义变量

###语法：变量名=数据

说明：a.变量名—-标识符（见名知义），并且不能是关键字

字母小写，多个字母用下划线隔开

b. = —-赋值符号，右边的数据赋值给左边的变量

c.数据 —-数据可以是不同类型的值，也可以是已经声明的变量

或者是运算表达式

已经声明过的变量可以重新赋值，新赋的值会覆盖之前的值

3.同时声明多个变量

4.Python中声明变量和给变量赋值的基本原理：

是先开辟空间将数据存起来，数据需要多大的内存就开辟多大的内存，然后再将变量指向对应的内存区域

Python支持的运算符：数学运算符,比较运算符,逻辑运算符,

赋值运算符

1.数学运算符：+(加)，-（减），*（乘），/（除）,//（整除）,%（取余）,**（幂运算）
